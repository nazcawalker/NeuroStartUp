# Домашнее задание по теме «Работа с удаленным репозиторием через GitHub»

1. Опубликовать свой проект 
2. Создать ветку в проекте
3. Слить изменения из одной ветки в другую

Данная практика поможет вам научиться демонстрировать свои проекты через GitHub, а также создавать копии чужих репозиториев и вносить изменения в данные копии проектов.
